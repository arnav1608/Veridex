# Veridex

Veridex is a claim reliability and discussion platform.

---

## 🚀 How to Run the Project

### 1️⃣ Clone the repository

git clone https://github.com/arnav1608/Veridex.git

cd Veridex

---

### 2️⃣ Run Backend

cd backend

npm install

npm start

---

### 3️⃣ Run Frontend

Go to frontend folder

Open index.html in browser

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB
- HTML, CSS, JavaScript
